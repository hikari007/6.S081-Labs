# MIT6.S081 Labs

6.S081是MIT的一门操作系统课程，课程基于一个类Unix但更简单的教学操作系统XV6来讲解，同时还配备了多个Lab，完成它们可以使你更加深刻地理解操作系统

更多细节请参考[6.S081: Operating System Engineering](https://pdos.csail.mit.edu/6.828/2020/index.html)
